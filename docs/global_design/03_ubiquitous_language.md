# ユビキタス言語一覧

## 👤 アカウント関連

| **用語** | **英語名** | **定義** |
|---------|-----------|----------|
| **ゲスト** | Guest | ログインしていないユーザー。ログイン画面のみ利用できる |
| **ユーザー** | User | Googleアカウントでログインしたユーザー。サブスクや支払い方法の管理ができる |
| **認証** | Auth | Googleログイン／ログアウトなど、ユーザーの本人確認機能 |
| **退会** | deleteAccount | アカウントと関連データ（サブスク、支払い方法）をすべて削除する操作 |

## 📝 サブスク関連

| **用語** | **英語名** | **定義** |
|---------|-----------|----------|
| **サブスク** | Subscription | ユーザーが契約しているサブスクリプションサービス1件 |
| **サブスクID** | SubscriptionId | サブスクを一意に識別するためのID |
| **サービス名** | ServiceName | サブスクの名称（例：Netflix、Spotify、Adobe CC） |
| **金額** | Amount | サブスクの支払い金額（円）。請求サイクルあたりの金額 |
| **請求サイクル** | BillingCycle | 支払いの周期。月額（monthly）または年額（yearly） |
| **基準日** | BaseDate | 請求日計算の基準となる日付。初回支払日または任意の請求日を登録 |
| **次回請求日** | NextBillingDate | 今日以降で最も近い請求日。基準日と請求サイクルから自動計算される（DBには保存しない） |
| **メモ** | Memo | サブスクに関する自由記述の補足情報 |
| **サブスク一覧** | SubscriptionList | 登録済みサブスクをカード形式で表示する画面 |

## 💳 支払い方法関連

| **用語** | **英語名** | **定義** |
|---------|-----------|----------|
| **支払い方法** | PaymentMethod | サブスクの支払いに使用するカードや決済手段（例：楽天カード、PayPal） |
| **支払い方法ID** | PaymentMethodId | 支払い方法を一意に識別するためのID |
| **支払い方法名** | PaymentMethodName | 支払い方法の表示名。ユーザーが自由に設定できる |
| **使用中チェック** | UsageCheck | 支払い方法がサブスクで使用されているか確認する処理。使用中なら削除不可 |
| **インライン登録** | InlineCreate | サブスク登録画面から直接、新しい支払い方法を追加する機能 |

## 💰 料金計算関連

| **用語** | **英語名** | **定義** |
|---------|-----------|----------|
| **月額換算** | MonthlyEquivalent | 年額サブスクを12で割って月額に換算した金額 |
| **年額換算** | YearlyEquivalent | 月額サブスクを12倍して年額に換算した金額 |
| **合計金額** | TotalAmount | 全サブスクの金額を合算した値。月額換算・年額換算の両方を表示 |
| **切り捨て** | Floor | 小数点以下を切り捨てる処理。円未満の端数処理に使用 |

## 🧭 操作・機能関連

| **用語** | **英語名** | **定義** |
|---------|-----------|----------|
| **登録** | Create | 新しいサブスクや支払い方法を作成する操作 |
| **編集** | Edit | 既存のサブスクや支払い方法を修正する操作 |
| **削除** | Delete | サブスクや支払い方法を削除する操作。物理削除（完全に消去） |
| **一覧表示** | List | 複数のサブスクや支払い方法を一覧で表示する画面 |
| **カード表示** | CardView | サブスクを1件ずつカード形式で表示するUIパターン |
| **確認ダイアログ** | ConfirmDialog | 削除などの操作前にユーザーに確認を求めるモーダル |

## 💾 データ・技術関連

| **用語** | **英語名** | **定義** |
|---------|-----------|----------|
| **データベース** | Database | Postgres（Neon）を使用してデータを保存 |
| **OAuth** | OAuth | Googleログインで使う認証の仕組み |
| **セッション** | Session | ログイン状態を維持するための仕組み。ログアウトで終了 |
| **物理削除** | HardDelete | データをDBから完全に削除する方式（論理削除の対義語） |
